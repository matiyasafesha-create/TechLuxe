##               TechLuxe
## This is over all look of what this TechLuxe Application will be implemented and what this particular project is doing.

1. This is an application that is showcasing the backend development of the application.
2. A RESTFULL backend built with spring boot JWT authentication and MySQL support user registration, login role based access control, product,shopping cat functionality and user profile management.
3. There are Features Like


* Authentication & Authorization 
* User Profiles 
* Products
* Shopping Cart
* Categories 
* Users 

- Authentication is where we give access for either our users or Admins. 
4. We have directories Configurations,data,MySQL,Models,Authentication,Security
5. These folders consist of classes interfaces and configuration dao.
6. Controllers have classes that will be taking a part on receiving front end requests made by the user and feed that to the methods.
7. MYSQL Dao Classes are data talks directly to MySQl databases which checks to see if we need to insert/get/delete/ those changes.
8. we also have interfaces that will implement some of the classes we have for the controller.
9. Object classes will describe what your application works with such as USER,PROFILE,PRODUCT,SHOPPING CART, SHOPPING CART ITEM.
